# 🌐 Abhishek Sahani — Developer Portfolio

A clean and responsive **personal portfolio website** built with HTML, CSS, and JavaScript — showcasing skills, projects, and contact info.

---

## ✨ Features

- 🌗 **Dark / Light Mode** — theme toggle with localStorage persistence
- ⌨️ **Typing Animation** — auto-cycling role titles in the hero section
- 📌 **Sticky Navbar** — with smooth scroll & active link highlighting
- 📱 **Fully Responsive** — mobile hamburger menu included
- 🚀 **Project Showcase** — cards with live preview links
- 📬 **Contact Form** — with client-side validation
- ⬆️ **Back to Top** button on scroll

---

## 🗂️ Sections

| Section | Description |
|---------|-------------|
| **Hero** | Profile photo, name, animated role titles, CTA buttons |
| **About** | Brief bio and background |
| **Skills** | Tech stack overview |
| **Projects** | Cards with description, tech tags & live links |
| **Contact** | Email, LinkedIn & contact form |

---


## 🛠️ Tech Stack

- **HTML5** — Structure
- **CSS3** — Styling, animations, dark mode, responsiveness
- **JavaScript** — Interactivity & DOM logic
- **Font Awesome** — Icons
